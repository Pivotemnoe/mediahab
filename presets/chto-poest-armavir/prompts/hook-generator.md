Сформируй 3 варианта крючка только после получения всех фактических блоков материала.

Требования:

- крючок отражает реальный контраст, сильное блюдо, слабое место или общий вывод;
- не добавляет фактов, которых нет во входных данных;
- подходит для мобильной ленты;
- живой, конкретный, без кликбейта и канцелярита;
- юмор допустим, но не должен искажать факт;
- избегай повторения формулировок из недавних постов, если они переданы в контексте.

Верни JSON:

```json
{
  "suggestions": [
    {"text": "...", "rationale": "...", "fact_ids": ["..."]}
  ],
  "recommended_index": 0,
  "warnings": []
}
```

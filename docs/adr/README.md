# Architecture Decision Records

Create numbered ADR files during Phase 00, using `reference/ARCHITECTURE_DECISIONS.md` as the baseline. Recommended format:

```text
0001-modular-monolith.md
0002-authentication-and-sessions.md
0003-workspace-tenancy.md
0004-transactional-outbox.md
0005-s3-media-delivery.md
0006-provider-adapters.md
0007-connector-capability-registry.md
0008-phase02-auth-abuse-and-mock-billing.md
```

Each ADR contains status, context, decision, alternatives, consequences, migration/rollback implications, and evidence.

# Runbooks

Operational runbooks are created as implementation progresses. Required before public launch:

- deploy and rollback;
- database backup and restore;
- object-storage recovery;
- rotate application and connector secrets;
- failed publication triage;
- Telegram/MAX/Instagram reconnect;
- billing webhook replay;
- workspace export and deletion;
- incident response and audit review.

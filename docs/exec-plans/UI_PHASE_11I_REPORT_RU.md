# UI Phase 11I — отчёт

## Что сделано

- Content Studio переведена с Telegram-first подачи на пользовательский `Мастер материала`.
- Добавлен шаблонный путь `Шаблон: Обзор места` с восемью шагами: место и адрес, медиа, атмосфера, блюда, итог, ИИ-блоки, версии платформ, публикация.
- На телефоне первым виден один основной путь: мастер материала и сбор материала; факты, превью, проверки, факт-локи и история оставлены в закрытых подробностях.
- На десктопе техническая студия сохранена: guided form, входные блоки, превью площадок, факт-локи и проверки остались доступны.
- Пользовательские тексты переименованы:
  - `Голосовой пилот` заменён на `Сбор материала`;
  - `Мастер и Telegram` заменён на `ИИ-сборка и версии`;
  - кнопка `Опубликовать в тестовый Telegram` оставлена только внутри блока `Площадка: Telegram`.
- Старт `/app/content/new` теперь говорит о создании материала по шаблону, а не о создании Telegram-черновика.
- Visual smoke расширен проверками нового мастера, восьми шагов, закрытых мобильных подробностей и сохранённой desktop-студии.
- В `docs/OPEN_QUESTIONS.md` добавлен вопрос о том, нужно ли закрепить `Обзор места` как конфигурируемое имя шаблона вместо старого имени рубрики `Обзор недели`.

# Контракт API

Базовый путь `/api/v1`, JSON в `snake_case`, даты ISO 8601 UTC. Каждая мутация проверяет workspace, роль и entitlements. Долгие операции возвращают `202` и ресурс задачи. OpenAPI генерирует FastAPI, а TypeScript-клиент строится из него.

## Общие правила

- Ошибка: `{ "error": { "code", "message", "details", "request_id" } }`.
- Optimistic locking через `version` или `If-Match`, конфликт — `409`.
- `Idempotency-Key` для публикаций, оплат, загрузок и отдельных AI-задач.
- Секреты никогда не возвращаются API.

## Основные группы маршрутов

### Авторизация

`/auth/register`, `/auth/login`, `/auth/logout`, `/auth/logout-all`, `/auth/refresh`, подтверждение email, восстановление пароля, `/me` и управление сессиями.

### Рабочие пространства

CRUD `/workspaces`, участники и приглашения. Чужой объект не раскрывается.

### Проекты

Список и создание, создание из пресета, импорт, экспорт, клонирование, архив, версии.

### AI-генератор проекта и рубрик

`/projects/generate-suggestions`, `/projects/{id}/rubrics/generate-suggestions`, принятие или удаление предложений. Предложение не становится активным без подтверждения.

### Рубрики

CRUD, архив, восстановление, клон, версии, схема формы и проверка JSON Schema.

### Правила, промпты и шаблоны

CRUD и история версий. Сырой редактор промпта может быть доступен только администраторам и по feature flag.

### Примеры

Ручное добавление, импорт JSON и Telegram, утверждение, отклонение, embeddings, переиндексация и статус import job.

### Контент и блоки

CRUD ContentItem, динамические поля, повторяемые группы, блокировка и разблокировка фактов.

### Голос и медиа

Presigned upload, подтверждение загрузки, удаление, запуск распознавания, статус, принятие и повтор.

### AI

Извлечение фактов, сборка master, крючок, оценки, quality check, платформенные версии, отмена и повтор задачи.

### Редакции

История, утверждение, fork, редактирование и валидация вариантов площадок.

### Подключения

Подключение OAuth или bot token, callback, проверка, refresh, отключение; направления проекта, capabilities и тест подключения.

### Публикации

Создание, планирование, немедленная отправка, отмена, повтор, обновление статуса, редактирование, удаление внешнего поста и история попыток. Публикация запрещена, если конкретная редакция варианта не утверждена.

### Биллинг

Тарифы, подписка, usage, checkout через mock, отмена, ручное назначение тарифа системным администратором, платежи и счета.

### Webhook

Отдельные endpoint для MAX, Meta, платежей и generic webhook. Обработчик быстро проверяет, дедуплицирует, сохраняет и передаёт обработку в фон.

Системные административные маршруты находятся под `/api/v1/admin` и требуют отдельной системной роли, а не обычного `workspace owner`.

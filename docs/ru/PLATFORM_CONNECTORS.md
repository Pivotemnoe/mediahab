# Коннекторы площадок

Коннектор — backend-код, а не только запись в базе. Он отвечает за авторизацию, медиа, форматирование, жёсткие лимиты, публикацию, статус, редактирование, удаление, ошибки и capabilities.

Общие классы ошибок: неверный токен, недостаток прав, rate limit, quota, неверный payload, неподходящее медиа, обработка медиа, временная недоступность, внешний объект не найден, операция не поддерживается.

## Telegram

Основной путь — `sendRichMessage` с rich HTML и `<tg-collage>`/`<tg-slideshow>`. Медиа передаются по HTTPS signed URL с достаточным TTL. На дату ТЗ: 32768 символов и 50 медиавложений. Fallback — `sendMediaGroup` 2–10 элементов и отдельный текст, только после явного подтверждения.

Обязательный live spike: fixture «У Доника», 7 фото, 3 видео, 4069 знаков, одна публикация; проверить mobile/desktop, edit и delete. Без ключей допускается только контрактный тест с честным статусом pending.

## MAX

Проверка текста до 4000, получение URL загрузки, загрузка каждого файла, ожидание готовности токена, `POST /messages`, сохранение ID. Использовать header Authorization, HTTPS webhook, быстрый ACK, не превышать 30 rps, повторять `attachment.not.ready` с backoff.

Официальный универсальный максимум количества вложений в просмотренной документации не указан. На этапе 00 протестировать смешанный набор из 10 файлов и сохранить результат как capability, не выдавая его за официальный лимит.

## Instagram

Профессиональный аккаунт, Meta OAuth, права и review. Режимы: изображение, видео, карусель 2–10, Reel. Caption максимум 2200, хэштеги 30, упоминания 20. Перед отправкой проверять текущую quota. Поток: signed URL, контейнеры, polling, parent-карусель, publish, permalink. Не повторять publish вслепую.

`manual_required`: неверный тип аккаунта, нет прав, закончилась quota, неподходящее медиа или исчерпан retry.

## Threads

Интерфейс подготовить, feature flag выключен. На дату ТЗ максимум 500 символов. Не считать Instagram-токены автоматически совместимыми.

## YouTube

Подготовить OAuth, выбор канала, метаданные, resumable upload, captions, comments/replies. Community draft остаётся ручным до подтверждения официального endpoint.

## Generic webhook

HTTPS endpoint, shared secret, подписанный JSON, idempotency key, media URL, безопасные повторы. Обязательная защита от SSRF и запрет private-network адресов.

## Manual export

Текст, форматированный вариант, JSON и manifest медиа без внешней авторизации.

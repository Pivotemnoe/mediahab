# API contract

Base path: `/api/v1`. JSON uses `snake_case`. Dates are ISO 8601 UTC. Pagination is cursor-based for activity-heavy resources and page-based only where stable administrative lists justify it. Every mutation requires workspace authorization and entitlement checks.

## Conventions

- Error envelope: `{ "error": { "code", "message", "details", "request_id" } }`.
- Optimistic locking: mutable resources accept `If-Match` or explicit `version` and return `409` on conflict.
- Idempotency: creation of publications, payments, uploads, and selected AI jobs accepts `Idempotency-Key`.
- Long-running jobs return `202` with a job resource.
- OpenAPI is generated by FastAPI; frontend client is generated from it.

## Administration

System operator routes are separated under `/api/v1/admin`, require a system role, and are never inferred from workspace `owner`.

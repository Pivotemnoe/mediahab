# Задание Codex — UI Phase 00: план и проверка фундамента

## Цель

Изучить основное ТЗ и весь UI reference pack, затем подготовить реализуемый план frontend без разработки продуктовых экранов.

## Что прочитать

1. Корневой `AGENTS.md`.
2. Каноническое английское ТЗ продукта.
3. `design/ui-reference/00_START_HERE_EN.md`.
4. Все документы в `design/ui-reference/docs/`.
5. Четыре HTML-концепта и PNG-референсы.

## Что сделать

- проверить текущую структуру frontend;
- найти противоречия;
- подготовить route map;
- подготовить component map;
- определить design tokens;
- определить mock service layer и fixtures;
- разделить server state, local draft state, form state и publication state;
- описать responsive;
- подготовить Playwright visual test plan;
- сформулировать точный объём UI Phase 01.

## Ограничения

Не строить весь UI, не копировать standalone HTML, не добавлять зависимости без подтверждения, не хардкодить название и рубрики.

В конце дать отчёт на русском и остановиться до подтверждения.

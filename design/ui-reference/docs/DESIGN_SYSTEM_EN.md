# Design System

## Stack

```text
Next.js
React
TypeScript
Tailwind CSS
shadcn/ui
Radix primitives
Lucide icons
React Hook Form
Zod
```

## Requirements

- Central CSS variable and Tailwind token system.
- Light editorial default theme.
- One primary accent color.
- Semantic success, warning, and danger colors.
- 12–20 px surface radii.
- Borders over heavy shadows.
- Medium information density.
- 16 px minimum input/editor text on mobile.
